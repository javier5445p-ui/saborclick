# SaborClick

Sistema web de reservas para restaurantes desarrollado con React.

## Descripción

SaborClick es una aplicación web que permite a los usuarios reservar mesas en un restaurante de forma rápida y sencilla.  
El sistema incluye registro de usuarios, inicio de sesión, reserva de mesas y una página de contacto con mapa de ubicación.

## Tecnologías utilizadas

- React
- JavaScript
- HTML
- CSS
- Node.js

## Instalación del proyecto

1. Clonar el repositorio

git clone https://github.com/javier5445p-ui/saborclick.git

2. Entrar al proyecto

cd saborclick

3. Instalar dependencias

npm install

4. Ejecutar la aplicación

npm start

La aplicación se ejecutará en:

http://localhost:3000

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Reserva de mesas
- Confirmación de reserva
- Página de contacto con mapa de ubicación
